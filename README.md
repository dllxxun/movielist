# Movie List Web Application

넷플릭스 스타일의 영화 목록 웹 애플리케이션입니다.

## 주요 기능

- TMDB API를 활용한 영화 정보 검색
- 장르별 영화 필터링
- 평점 기반 영화 정렬
- 최근 검색어 저장 기능
- 위시리스트 관리
- 다크/라이트 테마 지원

## 기술 스택

- Vue.js 3
- Vue Router
- Local Storage
- TMDB API

## 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/dllxxun/movielist.github.io.git

# 의존성 설치
npm install

# 개발 서버 실행
npm run serve
